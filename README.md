# 日麻工具

## 简介

实现了日麻点数计算以及练习功能。本项目采用 Vue 构建，核心逻辑使用 JavaScript 实现。

牌画素材来自 最完整的日本麻將中文維基百科。

支持对某些规则进行自定义（旁边的按钮）。现在可以自定义的规则有：食断，是否计多倍役满、复合役满，连凤雀头符数，是否计累计役满。

## 个人修改

添加了点数练习的历史记录功能。可以回顾错题。注意刷新页面会清除。如果有喜欢的题，记得导出json。
